20200318，V0.0.2
1、本次代码为客人3月13日测试的版本。
2、添加了主机上电不直接开机，必须要通过长按POWER KEY 开机的功能
  （若有适配器，按遥控器POWER KEY 也可以开，仅电池必须长按机器电源键才能开机）
3、添加了兼容两个遥控器的程序。5
4、更新WOOFER 和PR后的新版的调音。
5、关机后，等待4G模组发命令后再关机、断4G模组电。同时关机给关机等待提示灯。

20200319 v0.0.3
1、添加室内室外EQ切换功能。
2、IDLE状态、初始化时只开放POWER键，其他按键屏蔽。关机时，屏蔽所有按键。
3、更改模式切换顺序。 WIFI->BT->AUX.
4、AUX ,WIFI BT,FM 分别使用不同的增益（PCM1826）.


20200324：
1、添加TAS2825MUTE脚GPIO配置，实现MUTE功能控制。
2、添加主机检测控制盒是否接入功能。
3、将主机接受到的IR KEY 发送到控制盒，实现控制盒接入时，IR对FM的操作。
4、修改音频通道切换的逻辑（PCM1862不一直查询状态切换，而是在按键触发瞬间切换）；增加在音量为0时开启MUTE功能。
5、增加IR长按功能。
